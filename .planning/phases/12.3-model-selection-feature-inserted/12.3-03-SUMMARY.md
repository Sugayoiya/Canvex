# Phase 12.3 Plan 03: ModelSelector Component + API + Stores Summary

Reusable ModelSelector Pill+Popover component with modelsApi client, ProviderIcon relocation, and Zustand store extensions for chat/canvas model selection persistence.

## What Was Built

### Task 1: Relocate ProviderIcon + Add modelsApi + Extend Stores

**ProviderIcon relocation:**
- Copied full component to `web/src/components/common/provider-icon.tsx`
- Replaced admin version with re-export: `export { ProviderIcon } from "@/components/common/provider-icon"`
- All existing admin imports (provider-card, model-form-modal) continue working unchanged

**modelsApi added to `web/src/lib/api.ts`:**
- `AvailableModel` and `AvailableModelsResponse` TypeScript interfaces
- `modelsApi.getAvailable()` → GET /api/v1/models/available
- `modelsApi.getSystemDefaults()` → GET /api/v1/models/system-defaults
- `modelsApi.updateSystemDefaults()` → PATCH /api/v1/models/system-defaults
- Extended `usersApi` with `getSettings`/`updateSettings` for user-level model preferences
- Extended `teamsApi` with `getSettings`/`updateSettings` for team-level model preferences

**Chat store (`chat-store.ts`):**
- Added `selectedModelName: string | null` to state
- Added `setSelectedModel` action
- Wrapped store with `persist` middleware (name: `chat-model-selection`)
- `partialize` restricts persistence to `selectedModelName` only

**Canvas store (`canvas-store.ts`):**
- Added `nodeModelSelections: Record<string, string>` to state
- Added `setNodeModel(nodeId, modelName)` and `clearNodeModels()` actions
- `reset()` now clears `nodeModelSelections: {}`

**Commit:** c661820

### Task 2: Create ModelSelector Component (Pill + Popover)

Created `web/src/components/common/model-selector.tsx` (458 lines):

**Pill:**
- Two sizes: `sm` (28px height) and `md` (36px height)
- Shows ProviderIcon + display_name + ChevronDown chevron
- Null state: Sparkles icon + "自动选择"
- Loading state: skeleton pulse animation
- Disabled state: opacity 0.5, cursor not-allowed
- Open state: 1px border rgba(0,209,255,0.3)

**Popover:**
- Flat model list sorted by provider
- Width: 260px (sm), 300px (md), max-height 320px scrollable
- Conditional search input (>8 models): "搜索模型..." with Search icon
- Each item: selected indicator dot (4px, --ob-primary), ProviderIcon, display name, price
- Price formatting: $X.XX/M (per-million tokens), $X.XX/img, or "free"
- Empty state: "暂无可用模型" with admin contact sub-text

**Interaction:**
- Keyboard: ArrowDown/Up (wrapping), Enter (select), Escape (close)
- Click-outside: mousedown listener closes popover
- `requiredFeatures` prop filters models by ALL features present
- `modelType` prop: "llm" | "image" | "all"

**Accessibility:**
- `role="combobox"` + `aria-haspopup="listbox"` on pill
- `role="listbox"` on popover list
- `role="option"` + `aria-selected` on each item

**Commit:** 13101be
