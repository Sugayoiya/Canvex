# Phase 12.3 — UI Design Contract

> Visual and interaction contract for the Model Selection Feature phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (custom CSS variable tokens) |
| Preset | not applicable |
| Component library | none — inline style objects with `--cv4-*` / `--ob-*` CSS custom properties |
| Icon library | Lucide React |
| Font | Space Grotesk (headings, mono labels), Manrope (body, labels) |

Canvas-context components (ModelSelector in Chat/Canvas) use `--cv4-*` tokens.
Page-context components (Project detail, Settings pages) use `--ob-*` tokens.
Toast notifications use Sonner (already integrated).

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-text gap inside pill, inline spacing |
| sm | 8px | Pill internal padding (vertical), compact element gaps |
| md | 12px | Popover item padding, input area gaps |
| lg | 16px | Panel/section padding, popover container padding |
| xl | 24px | Section vertical spacing in settings pages |
| 2xl | 32px | Page-level section breaks |

Exceptions: Pill horizontal padding uses 12px (not 16px) to maintain the compact tag look established by existing `--cv4-radius-tag` pills in `ai-generate-panel.tsx`.

## Color

| Role | Token | Usage |
|------|-------|-------|
| Dominant (60%) | `--cv4-canvas-bg` / `--ob-surface-base` | Background surfaces |
| Secondary (30%) | `--cv4-surface-primary` / `--ob-glass-bg` | Pill background, popover container, cards |
| Accent (10%) | `--ob-primary` (#00D1FF) | Selected model indicator dot, active pill border on focus |
| Destructive | `--ob-error` (#FFB4AB) | Fallback failure toast icon only |

Accent reserved for:
- Selected model indicator dot (4px circle) in popover list
- Pill border when popover is open (`1px solid var(--ob-primary)` at 30% opacity)
- Fallback success toast accent bar (left-border, 2px)

Color does NOT apply to:
- Pill text (uses `--cv4-text-secondary` always)
- Popover item hover (uses `--cv4-hover-highlight`)
- Provider icons (retain their own brand colors)

### 8. Fallback Toast Notifications

Using Sonner (already installed and used in project).

**Fallback success (model downgraded):**
```typescript
toast.info("模型已自动切换", {
  description: `${requestedModel} 不可用，已切换到 ${actualModel}`,
  duration: 4000,
});
```

**Fallback exhausted (503):**
```typescript
toast.error("无可用模型", {
  description: "请联系管理员配置 API Key",
  duration: 6000,
});
```

No custom toast styling — use Sonner defaults which inherit the app theme.

## State Management

| Store | Field | Type | Persist |
|-------|-------|------|---------|
| `useChatStore` | `selectedModelName` | `string \| null` | Yes (localStorage via Zustand persist) |
| `useCanvasStore` | `nodeModelSelections` | `Record<string, string>` | No (per-session, cleared on canvas switch) |

Canvas model selections are per-node, per-session. Chat model selection is global and persistent across sessions.
