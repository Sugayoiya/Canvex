Implementation requirements:
- **Data hook:** `useQuery({ queryKey: ["models", "available"], queryFn: () => modelsApi.getAvailable().then(r => r.data), staleTime: 5 * 60 * 1000 })`
- **Filtering by modelType:** When `modelType="llm"` show only `data.llm`, when `"image"` show `data.image`, when `"all"` show `[...data.llm, ...data.image]`
- **Filtering by requiredFeatures (D-16):** After modelType filtering, if `requiredFeatures` is provided, further filter the list to only include models whose `features` array contains ALL strings in `requiredFeatures`. Example: `requiredFeatures={["llm", "image"]}` returns only models where `model.features.includes("llm") && model.features.includes("image")`. Implementation:
```typescript
const filteredModels = useMemo(() => {
  let models = modelType === "llm" ? data.llm
    : modelType === "image" ? data.image
    : [...data.llm, ...data.image];
  if (requiredFeatures?.length) {
    models = models.filter(m =>
      requiredFeatures.every(f => m.features.includes(f))
    );
  }
  return models;
}, [data, modelType, requiredFeatures]);
```
