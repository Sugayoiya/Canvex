## Fallback 行为与错误处理

### 模型不可用时

| Option | Description | Selected |
|--------|-------------|----------|
| 直接报错 | 返回明确错误，用户自行切换 | |
| 静默 fallback + toast | 自动降级 + 前端提示 | ✓ |
| 弹窗确认 | 弹窗让用户选择是否使用替代 | |

### Fallback 链全部耗尽

| Option | Description | Selected |
|--------|-------------|----------|
| 返回 503 | 明确错误 "无可用模型，请联系管理员" | ✓ |
| 你来定 | Claude 自行处理 | |

**User's choice:** 静默 fallback + toast；全链耗尽返回 503
**Notes:** 用户强调 fallback 后要持久化选择，避免重复触发 toast
