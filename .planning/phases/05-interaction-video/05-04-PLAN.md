**Shallow-equality guard (addresses HIGH review — prevent infinite render loops):**

The `useOnSelectionChange` hook calls onChange on every selection event. If the callback updates state that triggers a re-render which re-registers the callback, an infinite loop occurs. Solution: compare node ID arrays with shallow equality and only update state when they actually change.

```typescript
const prevSelectionRef = useRef<string[]>([]);

const onChange = useCallback(({ nodes }: { nodes: Node[] }) => {
  const newIds = nodes.map(n => n.id).sort();
  const prevIds = prevSelectionRef.current;

  // Shallow equality check — only update state when selection actually changes
  if (
    newIds.length === prevIds.length &&
    newIds.every((id, i) => id === prevIds[i])
  ) {
    return; // No change — skip state update to prevent re-render loop
  }

  prevSelectionRef.current = newIds;
  setSelectedNodes(nodes);
}, []); // Empty deps — uses ref for comparison, no closure variables

useOnSelectionChange({ onChange });
```

**Visibility-aware polling with error backoff (addresses MEDIUM review):**

```typescript
const pollRef = useRef<ReturnType<typeof setInterval> | null>(null);
const pollIntervalRef = useRef(3000); // Start at 3s
const BASE_POLL_INTERVAL = 3000;
const MAX_POLL_INTERVAL = 30000;

useEffect(() => {
  if (batchState !== "executing" || !batchId) return;

  const poll = async () => {
    // Pause when tab is hidden
    if (document.hidden) return;

    try {
      const res = await canvasApi.batchStatus(batchId);
      const { node_statuses, status } = res.data;
      const completed = Object.values(node_statuses).filter(
        (s: string) => ["completed", "failed", "blocked", "timeout"].includes(s)
      ).length;
      setCompletedCount(completed);

      // Reset backoff on success
      pollIntervalRef.current = BASE_POLL_INTERVAL;

      if (status === "completed" || status === "failed") {
        setBatchState("complete");
        clearInterval(pollRef.current!);
        pollRef.current = null;
      }
    } catch (err) {
      // Exponential backoff on error — double interval, cap at MAX
      pollIntervalRef.current = Math.min(pollIntervalRef.current * 2, MAX_POLL_INTERVAL);
      console.warn("Batch poll error, backing off to", pollIntervalRef.current, "ms");
    }
  };

  pollRef.current = setInterval(poll, pollIntervalRef.current);
  poll(); // Immediate first poll

  // Also listen to visibilitychange to resume polling when tab becomes visible
  const onVisibilityChange = () => {
    if (!document.hidden && batchState === "executing") {
      poll(); // Immediate poll on tab focus
    }
  };
  document.addEventListener("visibilitychange", onVisibilityChange);

  return () => {
    if (pollRef.current) clearInterval(pollRef.current);
    document.removeEventListener("visibilitychange", onVisibilityChange);
  };
}, [batchState, batchId]);
```

- `clearSelection`: resets selectedNodes to [], deselects in ReactFlow, resets prevSelectionRef
- `dismissBar`: sets batchState back to "idle"

Return: `{ selectedNodes, canBatchExecute, batchState, executeBatch, clearSelection, dismissBar, completedCount, totalCount }`
