**1. Create `api/app/services/graph_execution_service.py`:**

```python
def topological_sort(node_ids: set[str], edges: list[tuple[str, str]]) -> list[list[str]]:
```
- Kahn's algorithm BFS: build in_degree dict + adjacency list for selected subgraph only
- Return list of layers (list[list[str]]) where layer 0 = roots (in_degree 0)
- Raise `ValueError("Cycle detected in selected subgraph")` if processed count != len(node_ids)

```python
class BatchExecutionService:
    def __init__(self, db: AsyncSession, user):
        self.db = db; self.user = user

    async def start_batch(self, canvas_id: str, node_ids: list[str]) -> dict:
```
- Validate all node_ids belong to canvas_id (query CanvasNode WHERE id IN node_ids AND canvas_id = canvas_id)
- If any missing, raise HTTPException(400, f"Nodes not found in canvas: {missing}")
- Query CanvasEdge WHERE canvas_id AND source_node_id IN node_ids AND target_node_id IN node_ids
- Call topological_sort() → layers
- Generate batch_id = str(uuid4())
- Store batch state in-memory dict (module-level `_batch_store: dict[str, dict]`) keyed by batch_id:
  ```python
  _batch_store[batch_id] = {
      "canvas_id": canvas_id,
      "user_id": user.id,
      "layers": [[nid for nid in layer] for layer in layers],
      "node_statuses": {nid: "queued" for nid in node_ids},
      "current_layer": 0,
      "created_at": datetime.utcnow().isoformat(),
  }
  ```
- For layer 0 nodes: for each node, invoke the skill via `skillsApi.invoke` (or just mark as "queued" — actual execution dispatched by frontend per-node using existing `useNodeExecution`)
- Return `{"batch_id": batch_id, "layers": layers, "total_nodes": len(node_ids)}`


- Lookup _batch_store[batch_id] or raise 404
- Return `{"batch_id": batch_id, "layers": ..., "node_statuses": ..., "current_layer": ...}`

**3. Add endpoints to `api/app/api/v1/canvas.py`:**

```python
@router.post("/batch-execute", response_model=BatchExecuteResponse)
async def batch_execute(
    req: BatchExecuteRequest,
    user=Depends(get_current_user),
    db: AsyncSession = Depends(get_db),
):
    canvas = await _load_canvas(req.canvas_id, db)
    await resolve_project_access(canvas.project_id, user, db)
    svc = BatchExecutionService(db, user)
    return await svc.start_batch(req.canvas_id, req.node_ids)

@router.get("/batch-execute/{batch_id}", response_model=BatchStatusResponse)
async def get_batch_status(
    batch_id: str,
    user=Depends(get_current_user),
    db: AsyncSession = Depends(get_db),
):
    svc = BatchExecutionService(db, user)
    return await svc.get_batch_status(batch_id)

@router.patch("/batch-execute/{batch_id}/nodes/{node_id}")
async def update_batch_node_status(
    batch_id: str,
    node_id: str,
    status: str = Query(...),
    user=Depends(get_current_user),
    db: AsyncSession = Depends(get_db),
):
    svc = BatchExecutionService(db, user)
    await svc.update_node_status(batch_id, node_id, status)
    return {"ok": True}
```

- Per D-14: need time-series grouping on top of existing usage-stats
- Per pitfall 4 (SQLite DATE_TRUNC incompatibility): use dual-path grouping:
  ```python
  if settings.USE_SQLITE:
      fmt = {"hour": "%Y-%m-%d %H:00", "day": "%Y-%m-%d", "week": "%Y-%W"}
      date_group = func.strftime(fmt[granularity], AICallLog.created_at)

      date_group = func.date_trunc(granularity, AICallLog.created_at)
  ```

- Filter by start_date, end_date, optional project_id
- GROUP BY "period" ORDER BY "period"
- Return TimeSeriesResponse with granularity and points list
