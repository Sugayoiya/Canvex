## Design System

| Property | Value |

| Token | Value | Usage |
|-------|-------|-------|

## Interaction Contracts

### Canvas Selection → Batch Execution

| Step | Interaction | Visual Feedback |
|------|-------------|-----------------|
| 1 | User drags rectangle on canvas | Selection box with `--cv5-selection-fill/stroke` |
| 2 | Nodes inside rectangle become selected | Selected nodes: border 1.5px `var(--cv5-selection-stroke)`, shadow `var(--cv4-shadow-lg)` |
| 3 | `selectedNodes.length > 1` | BatchExecutionBar slides up from bottom (200ms ease-out) |
| 4 | Click "执行选中 (N)" | Bar transitions to executing state, nodes enter queued→running states |
| 5 | Each node completes | Node border flash green for 500ms, then returns to normal |
| 6 | Batch complete | Bar shows "完成" for 2s, then auto-dismiss |
| 7 | Click × on bar | Clears selection, bar dismissed |

### Toolbar Positioning V5

| Condition | Panel | Direction | Alignment | Gap |
|-----------|-------|-----------|-----------|-----|
| Image node, has content | ImageToolbar | above | center-aligned to node | 8px |
| Video node, has content | VideoToolbar | above | center-aligned to node | 8px |
| Audio node, has content | AudioToolbar | above | center-aligned to node | 8px |
| Text node, has content | TextToolbar (V4) | above | center-aligned to node | 8px |
| Any node, no content | AIGeneratePanel (V4) | below | left-aligned to node | 12px |

### Audio Playback

| Action | Result |
|--------|--------|
| Click play button | WaveSurfer starts playback, playhead moves, button → pause icon |
| Click pause | Playback pauses, playhead stops |
| Click on waveform | Seek to position |
| Click upload button | File picker for audio files (mp3, wav, ogg, m4a) |
| 2x toolbar button | Toggle playback rate: 1x → 1.5x → 2x → 0.5x → 1x |

### Admin vs Non-Admin Visibility

| Feature | Admin | Non-Admin |
|---------|-------|-----------|
| Billing KPIs | Global aggregates | Own usage only |
| Billing table | All users' data | Own records only |
| Task list user column | Visible, filterable | Hidden |
| Task list data | All tasks | Own tasks only |
| Node execution history | All executions | Own executions |

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| Chart alt text | Recharts `<desc>` element with summary: "使用趋势图：显示过去N天的费用和调用量" |
| Color contrast | All status badge text on badge backgrounds meets WCAG AA (4.5:1) |
