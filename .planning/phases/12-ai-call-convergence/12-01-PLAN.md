      - `async def report_error(self, key_id: str, error_type: str, error_message: str)` — pipeline: HINCRBY error_count, HSET last_used_at/last_error_type/last_error_at, LPUSH error entry to errors list, LTRIM 0..19
      - `async def reset_error_count(self, key_id: str)` — HSET error_count=0, DELETE errors list
      - `async def set_health(self, key_id: str, data: dict)` — HSET from dict (used during restore from DB)
      - `async def get_recent_errors(self, key_id: str, limit: int = 20) -> list[dict]` — LRANGE 0..limit-1, json.loads each
      - `async def get_usage_trend(self, key_id: str, hours: int = 24) -> list[dict]` — iterate last `hours` hourly keys, GET each counter, return `[{"hour": "YYYYMMDDHH", "count": int}]` sorted chronologically
      - `async def close(self)` — close Redis connection if open
