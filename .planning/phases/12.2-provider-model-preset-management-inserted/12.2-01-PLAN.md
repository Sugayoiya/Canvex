**1. Create api/app/services/ai/preset_seed.py** with:

```python
"""
Centralized preset seeding for Providers, Models, and Pricing.

Reads PROVIDER_META + _KNOWN_MODELS from each provider implementation.
Uses seed_version gating: only creates/updates records when code version > DB version.
Seed order: (1) providers, (2) models, (3) pricing associations.
"""
```

Implement these functions:

**`get_all_provider_metadata() -> list[dict]`:**
Import `PROVIDER_META` and `_KNOWN_MODELS` from `gemini`, `openai_provider`, `deepseek`.
Return list of `{**meta, "models": known_models}` for each.

**`async def seed_preset_providers(session: AsyncSession)`:**
- `CURRENT_SEED_VERSION = 1`
- For each provider metadata from `get_all_provider_metadata()`:
  - Query `AIProviderConfig` by `provider_name` + `owner_type="system"`
  - If exists and `seed_version >= CURRENT_SEED_VERSION`: skip (don't overwrite user changes)
  - If exists and `seed_version < CURRENT_SEED_VERSION`: update `description`, `icon`, `sdk_type`, `default_base_url`, `seed_version` (but NOT `display_name`, `is_enabled`, `priority`, `base_url` — those are user-editable)
  - If not exists: create new `AIProviderConfig` with `is_preset=True`, all meta fields, `owner_type="system"`, `priority` = index
- `await session.flush()` after all providers

**`async def seed_preset_models_and_pricing(session: AsyncSession)`:**
- `CURRENT_SEED_VERSION = 1`
- Collect all provider configs from DB (just-seeded) by provider_name
- For each provider metadata:
  - For each model in `models` dict:
    - Query `AIModelConfig` by `model_name`
    - If exists and `seed_version >= CURRENT_SEED_VERSION`: skip
    - If exists and `seed_version < CURRENT_SEED_VERSION`: update `display_name`, `input_token_limit`, `output_token_limit`, `seed_version`
    - If not exists: create `AIModelConfig` with `is_preset=True`, `model_name`, `display_name`, `model_type` (infer from `output_types`: if "image" in output_types → "image", else "llm"), `capabilities` (JSON string), `input_token_limit`, `output_token_limit`, `seed_version`
  - `await session.flush()` after models
  - For each model with `default_pricing`:
    - Look up the `AIModelConfig.id` and `AIProviderConfig.id`
    - Query `ModelPricing` by `provider_config_id + model_config_id` (FK match) OR by `provider + model` text match (backward compat)
    - If exists: update FK fields if null (`provider_config_id`, `model_config_id`), update pricing values only if seed version changed
    - If not exists: create `ModelPricing` with BOTH text fields (`provider=meta["provider_name"]`, `model=model_name`) AND FK fields (`provider_config_id`, `model_config_id`), plus pricing data from `default_pricing`
  - `await session.flush()` after pricing
