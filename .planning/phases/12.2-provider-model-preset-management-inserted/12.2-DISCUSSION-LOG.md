## 4a) base_url 用途

| Option | Description | Selected |
|--------|-------------|----------|
| 仅 API 代理/镜像 | 切换同一 Provider 的代理地址 | |
| OpenAI 兼容端点 | 支持第三方兼容 API | |
| 两者兼顾 | 默认值 + Admin 覆盖 | ✓ (refined) |
