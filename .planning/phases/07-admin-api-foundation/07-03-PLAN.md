  <action>
Create `api/app/api/v1/admin_observability.py` with:
- `router = APIRouter(prefix="/admin", tags=["admin-observability"])`
- `require_admin(user)` at top of every handler

**Endpoint A: `GET /teams`** (paginated, addresses review concern about missing pagination)
- Parameters:
  - `q: str | None = Query(None)` — search by team name via `ilike`
  - `limit: int = Query(20, ge=1, le=100)`
  - `offset: int = Query(0, ge=0)`
- Query all non-deleted teams from `Team`
- When `q` provided, filter: `Team.name.ilike(f"%{q}%")`
- Compute `total` count (separate count query with same filters)
- Return items containing: `id`, `name`, `description`, `created_at`, `member_count`
- Compute member count with a subquery or grouped join on `TeamMember`
- Return `AdminTeamListResponse` with `items`, `total`, `limit`, `offset`

**Endpoint B: `GET /dashboard`**
- Return a single object with fields:
  - `total_users` — `COUNT(User.id)`
  - `total_teams` — `COUNT(Team.id)`
  - `active_tasks` — `COUNT(SkillExecutionLog)` where `status in ["queued", "running"]`
  - `total_cost` — lifetime `SUM(AICallLog.cost)`
  - `provider_status` — `enabled_count` and `disabled_count` from `AIProviderConfig` **filtered to `owner_type='system'` ONLY**. This is the admin/system perspective; team/personal provider configs are irrelevant for the admin dashboard.
  - `windows` object with keys `h24`, `d7`, `d30`, each including:
    - `tasks_total` — COUNT of SkillExecutionLog in window
    - `tasks_failed` — COUNT where `status == "failed"` in window
    - `cost_total` — SUM of AICallLog.cost in window

Time windows are computed relative to `datetime.now(timezone.utc)` and each uses `created_at >= window_start`. Use conditional aggregation (`CASE WHEN ... END`) to compute all 3 windows in a single query per table for efficiency. This works on both SQLite and PostgreSQL (unlike `FILTER (WHERE ...)`).

Extend `api/app/schemas/admin.py` with response models:
- `AdminTeamListItem` (id, name, description, created_at, member_count)
- `AdminTeamListResponse` (items: list[AdminTeamListItem], total, limit, offset)
- `AdminDashboardWindowStats` (tasks_total, tasks_failed, cost_total)
- `AdminProviderStatus` (enabled_count, disabled_count)
- `AdminDashboardResponse` (total_users, total_teams, active_tasks, total_cost, provider_status: AdminProviderStatus, windows: dict with h24/d7/d30 keys)

Register router in `api/app/api/v1/router.py` as `admin_observability_router`.
