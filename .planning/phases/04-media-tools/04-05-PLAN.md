**1. Create web/src/components/canvas/panels/panel-host.tsx:**

PanelHost renders as an absolute overlay inside the ReactFlow container (NOT inside node components). It reads focus state and renders the correct panel at the correct position.

Key implementation details:
- Position is `absolute`, `z-50`, `pointer-events-auto`
- Uses `getPanelPosition()` from useNodeFocus to get screen coordinates
- For "above" panels: position the panel bottom-edge at the returned y coordinate using `transform: translateY(-100%)`
- For "below" panels: position the panel top-edge at the returned y coordinate
- Animate with opacity 0→1 + translateY(±4px) on 150ms ease-out (use CSS transition)
- Respect `prefers-reduced-motion: reduce` — no animation
- Click outside the panel + node dismisses (use mousedown listener on ReactFlow pane)

```typescript
interface PanelHostProps {
  focusedNodeId: string | null;
  panelType: "ai-generate" | "text-toolbar" | "template-menu" | null;
  panelDirection: "above" | "below";
  panelPosition: { x: number; y: number; width: number } | null;
  onBlur: () => void;
  onExecute: (skillName: string, params: Record<string, unknown>) => void;
}
```

Render conditionally:
- `panelType === "ai-generate"` → `<AIGeneratePanel />`
- `panelType === "text-toolbar"` → `<TextToolbar />`
- `panelType === "template-menu"` → `<TemplateMenu />`
