Create the provider infrastructure. All three LLM providers use the `openai` Python SDK (already in pyproject.toml) — Gemini uses the OpenAI-compatible endpoint, DeepSeek uses its OpenAI-compatible base_url.

**api/app/services/__init__.py** — empty init.

**api/app/services/ai/__init__.py** — re-export: `from .base import Message, LLMProviderBase` and `from .provider_manager import get_llm_provider`.

**api/app/services/ai/base.py:**
- `Message` dataclass with `role: str` ("system"|"user"|"assistant") and `content: str`.
- `LLMProviderBase` abstract class:
  - `provider_name: str`, `model_name: str`, `api_key: str`
  - `async def generate(self, messages: list[Message], **kwargs) -> tuple[str, dict]` — the public method. Calls `_generate_impl`, extracts usage from response, returns `(text, usage_dict)`. The `usage_dict` has keys `input_tokens` and `output_tokens`.
  - `async def _generate_impl(self, messages: list[Message], **kwargs) -> tuple[str, dict]` — abstract, subclasses implement. Returns `(generated_text, {"input_tokens": N, "output_tokens": M})`.
  - Constructor takes `model_name: str`, `api_key: str`.

**api/app/services/ai/provider_manager.py:**
- `get_llm_provider(provider_name: str | None = None, model_name: str | None = None) -> LLMProviderBase`
- If `provider_name` is None, auto-select: check `settings.GEMINI_API_KEY` → `settings.OPENAI_API_KEY` → `settings.DEEPSEEK_API_KEY`. Raise `ValueError("No AI provider API key configured")` if all empty.
- Cache instances in module-level `_provider_cache: dict[str, LLMProviderBase]` keyed by `"{provider}:{model}"`.
- Factory: `"gemini"` → `GeminiLLMProvider(model_name or "gemini-2.0-flash", settings.GEMINI_API_KEY)`, `"openai"` → `OpenAILLMProvider(model_name or "gpt-4o-mini", settings.OPENAI_API_KEY)`, `"deepseek"` → `DeepSeekLLMProvider(model_name or "deepseek-chat", settings.DEEPSEEK_API_KEY)`.
- `clear_cache()` function for testing.

**api/app/services/ai/providers/__init__.py** — re-export all 3 classes.

**api/app/services/ai/providers/openai_llm.py:**
- `class OpenAILLMProvider(LLMProviderBase)` with `provider_name = "openai"`.
- Constructor: creates `openai.AsyncOpenAI(api_key=api_key)` client.
- `_generate_impl`: converts `Message` list to `[{"role": m.role, "content": m.content}]`, calls `self.client.chat.completions.create(model=self.model_name, messages=msgs, **kwargs)`. Extracts `response.choices[0].message.content` as text. Extracts `response.usage.prompt_tokens` as `input_tokens` and `response.usage.completion_tokens` as `output_tokens`. Returns `(text, {"input_tokens": ..., "output_tokens": ...})`.

**api/app/services/ai/providers/deepseek_llm.py:**
- `class DeepSeekLLMProvider(LLMProviderBase)` with `provider_name = "deepseek"`.
- Constructor: creates `openai.AsyncOpenAI(api_key=api_key, base_url="https://api.deepseek.com")`.
- `_generate_impl`: identical to OpenAI implementation (DeepSeek is OpenAI-compatible).

**api/app/services/ai/providers/gemini_llm.py:**
- `class GeminiLLMProvider(LLMProviderBase)` with `provider_name = "gemini"`.
- Constructor: creates `openai.AsyncOpenAI(api_key=api_key, base_url="https://generativelanguage.googleapis.com/v1beta/openai/")`.
- `_generate_impl`: identical to OpenAI (Gemini OpenAI-compatible endpoint returns same response shape). Extracts usage the same way.
