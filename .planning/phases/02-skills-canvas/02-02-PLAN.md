**api/app/api/v1/canvas.py:**

Create a FastAPI APIRouter with prefix="/canvas" and tag "canvas". Follow the pattern in `skills.py` for auth dependency injection.

Endpoints:

1. `POST /` — Create canvas.
   - Depends: `get_current_user`, `get_db`.
   - Body: `CanvasCreate`. Set `created_by=user.id`.
   - Create `Canvas(...)`, `session.add()`, `await session.commit()`, `await session.refresh(canvas)`.
   - Return `CanvasListItem`.

2. `GET /` — List canvases for a project.
   - Query param: `project_id: str`.
   - Depends: `get_current_user`, `get_db`.
   - Query: `select(Canvas).where(Canvas.project_id == project_id, Canvas.is_deleted == False).order_by(Canvas.created_at.desc())`.
   - Return `list[CanvasListItem]`.

3. `GET /{canvas_id}` — Get canvas with nodes and edges.
   - Depends: `get_current_user`, `get_db`.
   - Query canvas, then query all CanvasNode and CanvasEdge where `canvas_id` matches.
   - Return `CanvasResponse` with populated `nodes` and `edges` lists.
   - 404 if canvas not found or is_deleted.

4. `PATCH /{canvas_id}` — Update canvas metadata.
   - Body: `CanvasUpdate`. Use `model_dump(exclude_unset=True)` + `setattr` pattern.
   - Return `CanvasResponse`.

5. `DELETE /{canvas_id}` — Soft-delete canvas.
   - Call `canvas.soft_delete()`, commit. Return `{"ok": True}`.

6. `POST /{canvas_id}/nodes` — Add node to canvas.
   - Body: `CanvasNodeCreate`.
   - Create `CanvasNode(canvas_id=canvas_id, **req.model_dump())`, add, commit, refresh.
   - Return `CanvasNodeResponse`.

7. `PATCH /{canvas_id}/nodes/{node_id}` — Update node.
   - Body: `CanvasNodeUpdate`. Use `model_dump(exclude_unset=True)` + `setattr`.
   - Return `CanvasNodeResponse`.

8. `DELETE /{canvas_id}/nodes/{node_id}` — Delete node.
   - Hard delete (not soft). Also delete any CanvasEdge where `source_node_id` or `target_node_id` matches.
   - Return `{"ok": True}`.

9. `POST /{canvas_id}/edges` — Add edge.
   - Body: `CanvasEdgeCreate`.
   - Validate both source_node_id and target_node_id exist in this canvas.
   - Create `CanvasEdge(canvas_id=canvas_id, **req.model_dump())`, add, commit.
   - Return `CanvasEdgeResponse`.

10. `DELETE /{canvas_id}/edges/{edge_id}` — Delete edge.
    - Hard delete. Return `{"ok": True}`.
