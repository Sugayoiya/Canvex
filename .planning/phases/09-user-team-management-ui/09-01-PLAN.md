    **AdminDataTable (`admin-data-table.tsx`):**

    Generic wrapper around `@tanstack/react-table` with server-side control.

    Props interface:
    ```typescript
    interface AdminDataTableProps<T> {
      table: Table<T>;           // TanStack Table instance from useReactTable
      isLoading?: boolean;       // Show skeleton rows
      isError?: boolean;         // Show error state
      onRetry?: () => void;      // Retry callback for error state
      emptyIcon?: React.ReactNode;
      emptyHeading?: string;
      emptyBody?: string;
      onClearFilters?: () => void; // "Clear filters" CTA in empty state
      skeletonWidths?: number[];   // Per-column skeleton bar widths
    }
    ```

    Implementation:
    - Outer container: `--cv4-surface-primary` bg, `1px solid --cv4-border-subtle` border, `borderRadius: 12`, `overflow: hidden`
    - Render `<table>` with `role="table"`, `width: 100%`, `borderCollapse: collapse`
    - `<thead>`: bg `--cv4-canvas-bg`, height 40px, border-bottom `--cv4-border-default`
      - Each `<th>`: `scope="col"`, padding `0 16px`, text: Space Grotesk 12px/700 uppercase `letterSpacing: "1px"` color `--cv4-text-muted`
      - If column `enableSorting`: render clickable header with `aria-sort` attribute. Show `arrow-up`/`arrow-down` (lucide 12px) after text. Sorted = `--cv4-text-primary`, unsorted = `--cv4-text-muted` at 50% opacity. Hover: `--cv4-text-secondary`.
    - `<tbody>`: iterate `table.getRowModel().rows`
      - Each `<tr>`: height 48px, border-bottom `1px solid --cv4-border-subtle` (not last), hover bg `#E5E2E108`, transition `background 100ms`
      - Each `<td>`: padding `0 16px`, use `flexRender(cell.column.columnDef.cell, cell.getContext())`
    - Loading state: when `isLoading`, render 10 skeleton rows (SkeletonRow) instead of data rows. Each skeleton: 48px height, pulsing rounded bars (12px height, `--cv4-text-muted` at 15% opacity, widths from `skeletonWidths` prop or defaults [120, 160, 60, 50, 80, 100, 100, 32])
    - Pulse animation: define CSS keyframes inline via a `<style>` tag inside the component: `@keyframes adminPulse { 0%,100% { opacity: 0.3 } 50% { opacity: 0.6 } }` with `animation: adminPulse 1.5s infinite`
    - Error state: when `isError`, render centered error display inside tbody (colspan full width): `alert-circle` icon 48px `--ob-error` 50% opacity, heading "Failed to load data" Space Grotesk 14px/700, body text, "Retry" button (primary style: `--cv4-btn-primary` bg, `--cv4-btn-primary-text` text, 36px height, 8px radius, 0 16px padding)
    - Empty state: when not loading/error and rows.length === 0, render centered empty display: icon (from prop, 48px, `--cv4-text-muted` 50% opacity), heading (Space Grotesk 14px/700), body (Manrope 12px/400 `--cv4-text-muted`), optional "Clear filters" button (secondary style: `--cv4-btn-secondary` bg, `--cv4-btn-secondary-text` text, `--cv4-btn-secondary-border` border)
