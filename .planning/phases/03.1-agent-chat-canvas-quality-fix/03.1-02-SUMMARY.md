## Decisions Made
- Used `localeCompare` on `edge.source` for deterministic ordering — simple, stable, and independent of edge creation order
- Replaced plan's `navigator.sendBeacon` with `fetch({ method: 'PATCH', keepalive: true })` in beforeunload handler — sendBeacon only sends POST (backend expects PATCH), lacks auth headers, and uses relative URLs (wrong host). `fetch` with `keepalive` provides the same unload survivability with proper method/headers/URL.
- Write serialization via `isSavingRef`/`needsSaveRef` — queues at most one pending save while a write is in-flight, preventing interleaved concurrent PATCH requests
