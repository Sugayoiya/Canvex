**2. Create `api/app/services/graph_execution_service.py`:**

```python
def topological_sort(node_ids: set[str], edges: list[tuple[str, str]]) -> list[list[str]]:
```
- Kahn's algorithm BFS: build in_degree dict + adjacency list for selected subgraph only
- **Orphan handling (addresses MEDIUM review):** Nodes with in_degree=0 AND out_degree=0 (no edges at all in subgraph) are placed in layer 0
- **Deterministic ordering (addresses MEDIUM review):** Sort each layer by node_id using `sorted()` before appending to layers list
- **Structured cycle errors (addresses MEDIUM review):** If processed count != len(node_ids), collect remaining (unprocessed) node IDs and raise `ValueError(f"Cycle detected among nodes: {sorted(remaining_ids)}")` — includes offending node IDs for user-facing error

```python
VALID_BATCH_STATUSES = {"queued", "running", "completed", "failed", "blocked", "timeout"}
TERMINAL_STATUSES = {"completed", "failed", "blocked", "timeout"}

class BatchExecutionService:
    def __init__(self, db: AsyncSession, user):
        self.db = db
        self.user = user

    async def start_batch(self, canvas_id: str, node_ids: list[str]) -> BatchExecution:
```
- Validate all node_ids belong to canvas_id (query CanvasNode WHERE id IN node_ids AND canvas_id = canvas_id)
- If any missing: raise HTTPException(400, f"Nodes not found in canvas: {sorted(missing)}")
- Query CanvasEdge WHERE canvas_id AND source_node_id IN node_ids AND target_node_id IN node_ids
- Call topological_sort() → layers (handles orphan nodes, deterministic ordering)
- Create `BatchExecution` row in DB:
  ```python
  batch = BatchExecution(
      canvas_id=canvas_id, user_id=user.id,
      layers=layers, node_statuses={nid: "queued" for nid in node_ids},
      current_layer=0, status="running", total_nodes=len(node_ids),
  )
  db.add(batch)
  await db.commit()
  await db.refresh(batch)
  ```
- Return batch object

- Query BatchExecution by id
- **Authorization check (addresses HIGH review):** Verify `batch.user_id == self.user.id` or user has project access via `resolve_project_access` on batch.canvas_id's project
- Raise 404 if not found, 403 if unauthorized
- Return batch

```python
    async def update_node_status(self, batch_id: str, node_id: str, status: str) -> BatchExecution:
```
- **Status validation (addresses MEDIUM review from 05-04):** Validate `status in VALID_BATCH_STATUSES`, raise 400 if invalid
- Query batch, **verify user authorization** (same as get_batch_status)
- **Verify node belongs to batch:** `if node_id not in batch.node_statuses: raise HTTPException(400, "Node not in batch")`
- **Validate transition:** Don't allow transitions FROM terminal states (completed/failed/blocked/timeout) — raise 409 Conflict
- Update node_statuses[node_id] = status in JSON
- If all nodes in current_layer are in TERMINAL_STATUSES, advance current_layer += 1
- If all nodes across all layers are terminal, set batch.status = "completed" (or "failed" if any failed)
- Commit and return updated batch

```python
class BatchExecuteRequest(BaseModel):
    canvas_id: str
    node_ids: list[str]

    @field_validator("node_ids")
    @classmethod
    def at_least_two(cls, v):
        if len(v) < 2:
            raise ValueError("Batch execution requires at least 2 nodes")
        return v

class BatchExecuteResponse(BaseModel):
    batch_id: str
    layers: list[list[str]]
    total_nodes: int

class BatchStatusResponse(BaseModel):
    batch_id: str
    layers: list[list[str]]
    node_statuses: dict[str, str]
    current_layer: int
    status: str

class BatchNodeUpdateRequest(BaseModel):
    status: str

    @field_validator("status")
    @classmethod
    def valid_status(cls, v):
        allowed = {"queued", "running", "completed", "failed", "blocked", "timeout"}
        if v not in allowed:
            raise ValueError(f"Status must be one of {allowed}")
        return v
```

**4. Add endpoints to `api/app/api/v1/canvas.py`:**

```python
@router.post("/batch-execute", response_model=BatchExecuteResponse)
async def batch_execute(
    req: BatchExecuteRequest,
    user=Depends(get_current_user),
    db: AsyncSession = Depends(get_db),
):
    canvas = await _load_canvas(req.canvas_id, db)
    await resolve_project_access(canvas.project_id, user, db)
    svc = BatchExecutionService(db, user)
    batch = await svc.start_batch(req.canvas_id, req.node_ids)
    return BatchExecuteResponse(batch_id=batch.id, layers=batch.layers, total_nodes=batch.total_nodes)

@router.get("/batch-execute/{batch_id}", response_model=BatchStatusResponse)
async def get_batch_status(
    batch_id: str,
    user=Depends(get_current_user),
    db: AsyncSession = Depends(get_db),
):
    svc = BatchExecutionService(db, user)
    batch = await svc.get_batch_status(batch_id)
    return BatchStatusResponse(
        batch_id=batch.id, layers=batch.layers,
        node_statuses=batch.node_statuses, current_layer=batch.current_layer,
        status=batch.status,
    )

@router.patch("/batch-execute/{batch_id}/nodes/{node_id}")
async def update_batch_node_status(
    batch_id: str,
    node_id: str,
    body: BatchNodeUpdateRequest,
    user=Depends(get_current_user),
    db: AsyncSession = Depends(get_db),
):
    svc = BatchExecutionService(db, user)
    await svc.update_node_status(batch_id, node_id, body.status)
    return {"ok": True}
```

Note: PATCH now uses a request body (`BatchNodeUpdateRequest`) instead of query param for status — more RESTful and enables Pydantic validation.
