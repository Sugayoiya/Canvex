**2. Create `api/app/api/v1/projects.py`:**

Router prefix `/projects`, tags `["projects"]`.

- `POST /` — Create project. Get current user. If `owner_type=="personal"`, set `owner_id=user.id`. If `owner_type=="team"`, validate team membership via `require_team_member(data.owner_id, user, db, "member")`. Set `created_by=user.id`. Use `model_dump(exclude_unset=True)` pattern.

- `GET /` — List projects. Accept `owner_type` and `owner_id` query params.
  - If `owner_type=="personal"`, filter `Project.owner_type=="personal" AND Project.owner_id==user.id`.
  - If `owner_type=="team"`, validate team membership first, then filter `Project.owner_type=="team" AND Project.owner_id==owner_id`.
  - Always filter `is_deleted==False`. Order by `created_at DESC`.
  - For each project, count canvases via subquery on Canvas table.
  - Return `list[ProjectResponse]`.

- `GET /{project_id}` — Get project detail. Use `resolve_project_access(project_id, user, db, "viewer")`. Count canvases.

- `PATCH /{project_id}` — Update. Use `resolve_project_access(project_id, user, db, "editor")`. Apply `model_dump(exclude_unset=True)` + `setattr` pattern.

- `DELETE /{project_id}` — Soft delete. Use `resolve_project_access(project_id, user, db, "admin")`. Call `project.soft_delete()`.

- `POST /{project_id}/clone` — Clone project (per D-23). Use `resolve_project_access(project_id, user, db, "viewer")` on source. Validate target ownership (personal: target_owner_id==user.id; team: require_team_member). Create new `Project` with same name+" (Copy)", description, global_style, aspect_ratio, new owner_type/owner_id. Clone canvases and their nodes/edges by reading originals and creating copies with new IDs.

```python
class TeamMemberQuota(Base):
    __tablename__ = "team_member_quotas"

    id: Mapped[str] = mapped_column(String(36), primary_key=True, default=lambda: str(uuid.uuid4()))
    team_id: Mapped[str] = mapped_column(String(36), index=True, nullable=False)
    user_id: Mapped[str] = mapped_column(String(36), index=True, nullable=False)
    monthly_credit_limit: Mapped[Decimal | None] = mapped_column(Numeric(12, 4), nullable=True)
    daily_call_limit: Mapped[int | None] = mapped_column(Integer, nullable=True)
    current_month_usage: Mapped[Decimal] = mapped_column(Numeric(12, 4), default=Decimal("0"))
    current_day_calls: Mapped[int] = mapped_column(Integer, default=0)
    last_month_reset: Mapped[datetime] = mapped_column(DateTime, default=datetime.utcnow)
    last_day_reset: Mapped[datetime] = mapped_column(DateTime, default=datetime.utcnow)
    created_at: Mapped[datetime] = mapped_column(DateTime, default=datetime.utcnow)
    updated_at: Mapped[datetime] = mapped_column(DateTime, default=datetime.utcnow, onupdate=datetime.utcnow)

    __table_args__ = (UniqueConstraint("team_id", "user_id", name="uq_team_member_quota"),)
```

This represents a team member's personal allocation from the team's total quota.
