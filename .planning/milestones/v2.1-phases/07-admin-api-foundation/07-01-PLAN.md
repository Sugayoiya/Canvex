Create `api/app/services/admin_audit.py` with a single helper:

```python
async def record_admin_audit(
    db: AsyncSession,
    *,
    admin_user_id: str,
    action_type: str,
    target_type: str,
    target_id: str,
    changes: dict,
    success: bool = True,
    error_message: str | None = None,
) -> AdminAuditLog:
```

Implementation rules:
- Serialize `changes` with `json.dumps(changes, ensure_ascii=False)`.
- Insert `AdminAuditLog` and `await db.flush()`.
- Return inserted row.
- Keep helper side-effect free beyond row creation.

Create `api/app/schemas/admin.py` with:
- `AdminAuditLogResponse` (all model fields, `changes` as dict)
- `AdminMutationResult` (generic `ok`, `message`, `audit_id` for mutation APIs)
- `AdminListMeta` (pagination metadata: `total`, `limit`, `offset`)

Use `model_config = {"from_attributes": True}` where needed.
